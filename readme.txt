Boostrap Basics, Utilities, Forms, and Components 

This assignment was to create a webpage using Bootstrap.

This webpage includes a navigation bar with the following sections: 
HOME, ABOUT, CONTACT. 
When displaying webpage on smaller screens, coding will auto collapse
navigation bar into a hamburger menu.

Webpage includes a registration form with the following components: 
First name, last name, email and password. It also has a check box to agree
to terms and conditions with a Submit button. 

Next section contains a table to display data. The data table contains
sample data of what will be collected in the registration form once entered
and terms and conditions are accepted. Code has striped and hover features for better readability

Main image and secondary image with round feature.
